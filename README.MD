# STL Containers.

Own implementation of STL Containers. 
The program is developed in C++ using the C++17 standard.

### Implemented containers:
1. list
2. map
3. set
4. Red Black Tree for map and set
5. array
6. queue
7. stack
8. vector

### Usage:
1. Clone repository:
```sh
git clone https://github.com/Reeskoos/stl-containers
```
2. Use Makefile or run CMake directly:
```sh
mkdir -p build
cd build
cmake -DCMAKE_BUILD_TYPE=Coverage ..
cmake --build .
```
